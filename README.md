# My Python Internship Projects

Welcome! This repository contains the Python projects I am building during my internship. It includes two useful command-line (CLI) tools.

---

## 📂 Folders in this Repository

Here is what you will find inside this workspace:

* **`github-user-activity/`**: A tool that connects to the GitHub API to fetch and display what a specific user has been up to recently (like their latest commits or stars).
* **`task-tracker/`**: A personal task manager app. It saves your tasks to a local file (`.json`) so you can add, update, track, and delete your to-do lists directly from your terminal.
