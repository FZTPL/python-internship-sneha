# My Python Internship Projects

Welcome! This repository contains the Python projects I am building during my internship. These projects focus on strengthening my understanding of Python fundamentals, file handling, APIs, JSON processing, command-line applications, and web development.

---

## 📂 Projects in this Repository

### 📌 `github-user-activity/`

A CLI application that connects to the GitHub API and displays a user's recent activity, such as repository events, commits, stars, and other public actions.

**Skills:** API Integration, JSON Handling, HTTP Requests, CLI Development

---

### 📌 `task-tracker/`

A command-line task management application that allows users to create, update, delete, and track tasks. Data is stored locally using JSON files.

**Skills:** CRUD Operations, File Handling, JSON Storage, CLI Development

---

### 📌 `expense-tracker/`

A command-line expense management application that helps users record, update, delete, and analyze expenses. The application also provides expense summaries and monthly spending reports.

**Skills:** File Handling, JSON Storage, CRUD Operations, Data Processing, CLI Development

---

### 📌 `number-guessing-game/`

An interactive CLI game where players attempt to guess a randomly generated number within a limited number of attempts based on selected difficulty levels. Includes hints, timers, and replay functionality.

**Skills:** Python Logic Building, Loops, Conditionals, Random Module, User Input Handling

---

### 📌 `personal-blog/`

A web-based blogging platform built with Flask that allows users to create, view, edit, and delete blog posts through a simple and user-friendly interface.

**Skills:** Flask, Web Development, CRUD Operations, Templates, Routing
